What is it

把"运行时"当成产品，而不是又一个 API 套壳

过去的 AI 工具大多围着"提示词"和"界面"打转。Hermes 把重心挪到了真正难的地方——记忆纪律、编排、故障恢复与长期运营。它由 Nous Research 于 2026 年 2 月 25 日发布，是首个完全 MIT 授权的"Tier 3 运行时"。

它住在你的服务器上，而不是绑在你的笔记本或某个 IDE 里。你可以在云端 VM 上让它干活，同时用手机上的 Telegram 跟它对话——上下文不会断。

它把"学习"落成了看得见的文件:技能、记忆、日志,而不是藏在不可见的隐状态里。多智能体协作被设计成可持久的编排,而不是花哨的表演式 swarm。

一句话:它把持续工作的能力——记得住、恢复得了、协调得动——当作核心,而不是聊得有多溜。

Get started

无前置依赖 · 两步上手

支持 Linux、macOS 与 WSL2,所有依赖自动安装。

1安装一条 curl 命令,搞定全部环境
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
2配置连接你的消息账号与模型供应商
hermes setup
原生 Windows 支持仍处于实验阶段。建议安装 WSL2,并在其中运行 Hermes Agent,以获得稳定体验。

把它装上,然后让它替你越变越能干。
