// Installation

三步完成安装

支持 macOS、Linux 与 Windows，也提供 Docker 镜像与 IDE 插件。

复制 
# 使用 Homebrew 安装（推荐）
brew install codegraph

# 或使用 curl 脚本
curl -fsSL https://codegraph.dev/install.sh | sh

# 验证安装
codegraph --version
CodeGraph v2.1.0 — build 2026.06.01
i

macOS 10.15+ 支持原生 Apple Silicon 与 Intel 芯片。首次运行如遇安全提示，请前往「系统偏好设置 → 安全性与隐私」允许。

复制 
# Debian / Ubuntu
wget -qO- https://codegraph.dev/deb/codegraph.asc | sudo apt-key add -
echo "deb https://codegraph.dev/deb stable main" | sudo tee /etc/apt/sources.list.d/codegraph.list
sudo apt update && sudo apt install codegraph

# Arch Linux
yay -S codegraph

# 验证
codegraph --version
i

支持 x86_64 与 ARM64 架构。若使用其他发行版，可从 GitHub Releases 下载预编译二进制文件。

复制 
# PowerShell (推荐)
winget install CodeGraph.CodeGraph

# 或使用 Scoop
scoop install codegraph

# 验证
codegraph --version
i

Windows 10 19041+ / Windows 11 支持。安装后会自动加入 PATH，重启终端即可使用 codegraph 命令。

复制 
# 拉取镜像
docker pull codegraph/codegraph:latest

# 扫描当前目录并启动 Web 服务
docker run --rm -p 8080:8080 -v "$(pwd):/project" codegraph/codegraph serve /project

# 访问 http://localhost:8080 查看图谱
i

Docker 镜像内置完整运行环境，适合 CI/CD 流水线中自动生成并部署图谱报告。

// Quick Start

快速上手

从初始化到深入探索，五分钟掌握 CodeGraph 的核心工作流。

1

初始化项目扫描 必须

进入项目根目录，运行 scan 命令。CodeGraph 会自动识别项目类型（Node.js、Python、Go、Rust 等）并解析源码结构。

$ codegraph scan --depth 3
✓ Detected Node.js project (package.json)
✓ Parsing 1,247 files... done (0.8s)
✓ Building dependency graph... done
✓ Graph saved to .codegraph/graph.json
2

启动可视化界面 推荐

serve 命令会在本地启动一个交互式 Web 界面，自动打开浏览器，支持实时热更新。

$ codegraph serve
🚀 CodeGraph server running at http://localhost:8080
📝 Watching for file changes...
⌨️ Press q to quit, h for help
3

交互式探索图谱

在浏览器中，你可以缩放拖拽整个图谱，点击节点查看详情，使用搜索框定位模块，或者通过筛选器按文件类型/依赖层级缩小范围。

4

导出与分享 可选

将图谱导出为静态文件或生成在线分享链接，方便在文档、PR 或会议中引用。

$ codegraph export --format svg --output architecture.svg
$ codegraph share --expires 7d
🔗 Share link: https://codegraph.dev/s/x8k2m9p (expires in 7 days)
// Commands

核心命令一览

简洁一致的 CLI 设计，所有命令遵循 codegraph <verb> [options] 范式。

❖
输入命令或按 ↑↓ 导航
ESC 关闭
🔍
scan
扫描项目并生成依赖图谱
codegraph scan
🚀
serve
启动本地可视化服务器
codegraph serve
📤
export
导出图谱为 SVG / PNG / JSON
codegraph export
🔗
share
生成可分享的在线链接
codegraph share
🧹
clean
清除缓存与生成的图谱文件
codegraph clean
⚙️
config
查看与修改配置项
codegraph config
