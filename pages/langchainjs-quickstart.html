Step 1

安装与配置

LangChain.js 支持 Node.js 18+ 和现代浏览器环境。推荐使用 TypeScript 获得最佳体验。

创建项目

terminal 
mkdir my-langchain-app && cd my-langchain-app
npm init -y
npm install -g typescript tsx
npx tsc --init

安装核心包

terminal 
# 核心库
npm install langchain

# OpenAI 集成（最常用）
npm install @langchain/openai

# 可选：其他模型提供者
npm install @langchain/anthropic    # Claude
npm install @langchain/google-genai  # Gemini
提示

运行环境变量请使用 dotenv：在项目根目录创建 .env 文件存放 API Key，不要提交到 Git。

.env 
OPENAI_API_KEY=sk-your-key-here
Step 2

核心概念

LangChain.js 的设计围绕六大核心模块。理解它们，你就理解了整个框架。

🔗

Chains

将多个调用串联成完整的工作流，从前处理到后处理一气呵成。

🧠

Models

统一的 LLM 和 ChatModel 接口，支持 OpenAI、Anthropic、Google 等数十家。

📄

Prompt

提示词模板系统，支持变量注入、消息历史、部分填充。

💾

Memory

对话记忆管理，让模型"记住"上下文，实现多轮对话。

🔍

Retrievers

向量检索，从外部知识库中找到最相关的信息片段。

🤖

Agents

让模型自主决策工具调用，完成复杂的多步骤任务。

架构关系一览

┌─────────────────────────────────────────┐ │ 🤖 Agent │ │ (决策层 — 决定调用什么工具) │ └────────┬──────────────┬────────────────┘ │ │ ┌────▼────┐ ┌─────▼──────┐ │ 🔗 Chain │ │ 📄 Retriever│ │(编排层) │ │ (检索层) │ └────┬────┘ └─────┬──────┘ │ │ ┌────▼────────────────▼─────┐ │ 🧠 Model (模型层) │ │ ChatModel / LLM / Embed │ └──────────────┬────────────┘ │ ┌──────────────▼────────────┐ │ 💾 Memory (记忆层) │ │ Buffer / Summary / Window │ └───────────────────────────┘
Step 3

构建你的首个 Chain

用最少的代码体验 LangChain.js 的核心流程：Prompt → Model → Output。

1

引入模块

从对应包中导入所需的 Chat Model 和 Prompt 模板。

index.ts 
import { ChatOpenAI } from "@langchain/openai";
import { PromptTemplate } from "@langchain/core/prompts";
import { StringOutputParser } from "@langchain/core/output_parsers";
import { RunnableSequence } from "@langchain/core/runnables";
2

定义 Prompt 模板

使用 PromptTemplate 创建可复用的提示词，支持动态变量。

index.ts 
const prompt = PromptTemplate.fromTemplate(
  "你是一位资深技术博主。请用通俗易懂的方式，向初学者解释以下概念：\n\n{topic}\n\n要求：使用类比，不超过 200 字。"
);
3

用 LCEL 编排 Chain

LangChain Expression Language (LCEL) 使用 pipe() 优雅地串联组件。

index.ts 
// 初始化模型
const model = new ChatOpenAI({
  modelName: "gpt-4o-mini",
  temperature: 0.7,
});

// 用 pipe 串联：模板 → 模型 → 解析器
const chain = prompt.pipe(model).pipe(new StringOutputParser());

// 调用！
const result = await chain.invoke({
  topic: "什么是 LangChain？"
});

console.log(result);
4

运行

terminal 
npx tsx index.ts
Step 4

RAG：检索增强生成

RAG 让 LLM 基于你自己的数据进行回答，是生产环境最常用的模式。

📑

Load

加载文档（PDF、网页、数据库）

✂️

Split

文档切分为合适大小的 Chunk

🔎

Retrieve

向量化后检索相关片段

rag.ts 
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";
import { RecursiveCharacterTextSplitter } from "langchain/text_splitter";

// 1. 加载 & 切分文档
const text = "你的长文档内容...";
const splitter = new RecursiveCharacterTextSplitter({
  chunkSize: 1000,
  chunkOverlap: 200,
});
const docs = await splitter.createDocuments([{ pageContent: text }]);

// 2. 创建向量存储
const vectorStore = await MemoryVectorStore.fromDocuments(
  docs,
  new OpenAIEmbeddings()
);

// 3. 创建检索器
const retriever = vectorStore.asRetriever(3); // 取 top-3

// 4. 组合成 RAG Chain
import { createRetrievalChain } from "langchain/chains/retrieval";
import { ChatOpenAI } from "@langchain/openai";
import { createStuffDocumentsChain } from "langchain/chains/combine_documents";
import { PromptTemplate } from "@langchain/core/prompts";

const answerPrompt = PromptTemplate.fromTemplate(
  "根据以下上下文回答问题。如果上下文中没有答案，请如实说明。\n\n上下文：{context}\n\n问题：{input}"
);

const model = new ChatOpenAI({ modelName: "gpt-4o-mini" });
const questionAnswerChain = await createStuffDocumentsChain({
  llm: model,
  prompt: answerPrompt,
});
const ragChain = await createRetrievalChain({
  retriever,
  combineDocsChain: questionAnswerChain,
});

// 5. 查询！
const response = await ragChain.invoke({
  input: "文档里讲了什么？"
});
console.log(response.answer);
生产建议

生产环境请使用持久化向量数据库（Pinecone、Weaviate、Chroma）替代 MemoryVectorStore。LangChain.js 支持数十种向量库集成。

Step 5

Agents：让模型自主行动

Agent 根据用户问题自主选择并调用工具，动态决定下一步操作。

Agent 与 Chain 的区别？

Chain 的执行路径是预先定义的，你明确知道每一步做什么。Agent 的执行路径是动态决定的 — 模型根据上下文自主选择调用哪些工具、以什么顺序。简单说：Chain 是固定流程，Agent 是 AI 驱动的决策引擎。

agent.ts 
import { ChatOpenAI } from "@langchain/openai";
import { tool } from "@langchain/core/tools";
import { z } from "zod";
import { createReactAgent } from "@langchain/langgraph/prebuilt";

// 定义工具
const weatherTool = tool(
  async ({ city }) => {
    // 在真实场景中，这里调用天气 API
    return `${city} 今天晴天，25°C`;
  },
  {
    name: "get_weather",
    description: "获取指定城市的天气信息",
    schema: z.object({
      city: z.string().describe("城市名称"),
    }),
  }
);

const searchTool = tool(
  async ({ query }) => {
    // 在真实场景中，这里调用搜索 API
    return `搜索 "${query}" 的结果：...`;
  },
  {
    name: "search",
    description: "搜索互联网上的信息",
    schema: z.object({
      query: z.string().describe("搜索关键词"),
    }),
  }
);

// 创建 Agent
const model = new ChatOpenAI({ modelName: "gpt-4o-mini" });
const agent = createReactAgent({
  llm: model,
  tools: [weatherTool, searchTool],
});

// 运行
const result = await agent.invoke({
  messages: [
    { role: "user", content: "北京今天天气怎么样？需要带伞吗？" }
  ]
});

console.log(result.messages[result.messages.length - 1].content);

对话记忆

memory.ts 
import { ChatOpenAI } from "@langchain/openai";
import { BufferMemory } from "langchain/memory";
import { ConversationChain } from "langchain/chains";

const model = new ChatOpenAI({ modelName: "gpt-4o-mini" });
const memory = new BufferMemory();
const chain = new ConversationChain({ llm: model, memory });

// 第一轮
const r1 = await chain.invoke({ input: "我叫小明" });
// 第二轮 — 模型会记得你叫小明
const r2 = await chain.invoke({ input: "我叫什么名字？" });
console.log(r2.response); // "你叫小明"

结构化输出 (Zod Schema)

structured.ts 
import { ChatOpenAI } from "@langchain/openai";
import { z } from "zod";

const model = new ChatOpenAI({ modelName: "gpt-4o-mini" });
const structuredModel = model.withStructuredOutput(
  z.object({
    sentiment: z.enum(["positive", "negative", "neutral"]),
    confidence: z.number().min(0).max(1),
    summary: z.string(),
  })
);

const result = await structuredModel.invoke(
  "分析这段话的情感：今天天气真好，心情愉快！"
);
// { sentiment: "positive", confidence: 0.95, summary: "..." }

流式输出

stream.ts 
import { ChatOpenAI } from "@langchain/openai";
import { PromptTemplate } from "@langchain/core/prompts";
import { StringOutputParser } from "@langchain/core/output_parsers";

const chain = new PromptTemplate.fromTemplate("给 {topic} 写一句标语")
  .pipe(new ChatOpenAI({ modelName: "gpt-4o-mini", streaming: true }))
  .pipe(new StringOutputParser());

const stream = await chain.stream({ topic: "AI 教育" });
for await (const chunk of stream) {
  process.stdout.write(chunk);
}
// 逐字输出：让 每 个 人 都 能 拥 有 A I 助 手 ...
